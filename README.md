# OnePDF+ Privacy Policy

Public privacy policy for the **OnePDF+** Android app (`com.onepdf.plus`).

## Live URL

After GitHub Pages is enabled:

**https://arjunr7.github.io/onepdf-plus-privacy/**

Use this URL in Google Play Console → App content → Privacy policy.

## Setup (one time)

1. Repository is **public** (required for Play Store).
2. Go to **Settings → Pages**.
3. **Source:** Deploy from branch `main` → folder `/ (root)`.
4. Save — site goes live in 1–2 minutes.

## Updating

Edit `index.html`, commit, and push to `main`.
